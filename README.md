# 뉴스 워드클라우드 생성기

뉴스 기사 콘텐츠를 수집해 단어 빈도 분석과 워드클라우드 시각화를 제공하는 Flask 기반 웹 애플리케이션입니다.

## 주요 기능

### 1. 뉴스 링크 입력
- 네이버 뉴스 기사 URL을 입력하면 본문을 자동으로 불러와 분석합니다.

### 2. 텍스트 업로드 및 전처리
- `.txt` 파일을 업로드해 직접 확보한 기사 전문도 분석할 수 있습니다.
- 불필요한 특수문자와 숫자를 제거하고 한글·영문 단어만 남겨 정제합니다.
- 기본 불용어 목록을 적용해 의미 없는 조사·접속사를 필터링합니다.

### 3. 단어 빈도 분석
- 정제된 단어의 출현 빈도를 계산해 상위 10개 키워드를 도출합니다.
- `collections.Counter`를 활용해 빠르게 빈도를 집계합니다.
- 분석 결과를 막대 차트와 워드클라우드 데이터로 변환합니다.

### 4. 시각화 대시보드
- **막대 그래프**: Chart.js를 이용해 단어 빈도수를 직관적으로 표현합니다.
- **워드클라우드**: WordCloud2.js로 키워드의 상대적 중요도를 한눈에 파악합니다.
- **반응형 UI**: 다양한 화면 크기에서 동일한 경험을 제공하도록 구성했습니다.

## 기술 스택

- **Backend**: Flask (Python)
- **웹 스크래핑**: BeautifulSoup, requests
- **데이터 처리**: re, collections.Counter
- **Frontend**: HTML, CSS, JavaScript
- **차트 라이브러리**: Chart.js, WordCloud2.js

## 설치 및 실행

### 필수 패키지 설치
```bash
pip install flask requests beautifulsoup4
```

### 실행
```bash
python app.py
```

브라우저에서 `http://localhost:5000` 접속

## 사용 방법

1. **홈 페이지**: 서비스 소개와 입력 옵션을 확인합니다.
2. **데이터 입력**: 뉴스 링크를 제출하거나 `.txt` 파일을 업로드해 분석을 요청합니다.
3. **결과 페이지**: 상위 10개 단어의 막대 그래프와 워드클라우드를 확인하고 인사이트를 도출합니다.
